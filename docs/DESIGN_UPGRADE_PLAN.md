# EchoVerse Homepage - Design Upgrade Plan

## Review Summary

After reviewing all pages, here are the key areas identified for improvement:

### Issues Found

1. **Inconsistencies:**
   - Downloads page title says "Tools" (wrong)
   - Downloads page has both "Downloads" and "Tools" marked active
   - Listen page has compressed/minified CSS (harder to maintain)
   - Thank-you page has different header structure (back button vs logo)
   - Inconsistent animations across pages
   - Footer inconsistencies

2. **Design Improvements:**
   - Standardize header/navigation across all pages
   - Add consistent page load animations
   - Improve hover states and transitions
   - Better spacing and visual hierarchy
   - Enhanced typography consistency
   - Better mobile responsiveness

3. **UX Enhancements:**
   - Better focus states for accessibility
   - Smoother transitions
   - More polished interactions
   - Better visual feedback

## Implementation Details

### Standardized Header Pattern
All pages should have:
- Logo with icon overlay
- Brand name + subtitle
- Consistent navigation
- Service status chips (where applicable)

### Animation System
- Page load: fade-in (0.4s)
- Header: slide-down (0.4s)
- Cards: staggered slide-up (0.15s, 0.25s delays)
- Hover: smooth transitions (0.2s)

### Typography Scale
- H1: 38-44px (page dependent)
- H2: 18-22px
- Body: 13-15px
- Small: 12px

### Spacing System
- Section padding: 28-40px vertical
- Card padding: 16-18px
- Gap between elements: 10-14px
- Margin top for sections: 18px
