Definition of Done

If these aren’t true, it stays internal. Simple rules, no excuses.

EchoVerse Done Criteria - No dead clicks: all unwired links are blocked with modal - Consistent navigation across all pages - Vault layout defined (packs/stems/bundles) + copyable paths - Catalog snapshot plan exists (inventory + diff + missing art) - Tool distribution plan selected (zip / releases / containers) - Optional: deep-link scheme defined (track_id / bundle_id)
