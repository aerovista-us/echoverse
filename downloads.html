Vault Structure (Staged)

You can ship tools in three ways later. For now, the UI stays stable while you decide the cleanest pipeline.

Option A — Internal ZIP Vault
Fastest: publish zips to a Vault folder and link them here.
Option B — GitHub Releases
Cleaner public future: attach binaries/zips to tagged releases.
Option C — Container Images
Ops-friendly: tools run as containers with mounted volumes.
Wiring Rule
Until a delivery path is picked, links stay blocked — no dead clicks.
